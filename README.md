# TarikTaliMatematik
A Malaysian heritage tug-of-war game reimagined into a real-time math battle that makes kids learn while they play.

Heritage Reimagined Educational Game

Built using Gemini Canvas
---

# Overview

Tarik Tali Matematik is a web-based educational game inspired by Malaysia’s traditional tug-of-war game.

It combines:
- gamified learning
- mathematics training
- cultural heritage
- multiplayer interaction

Players solve math questions to pull a rope and win.

---

# Game Mechanics

Correct answer:
- pulls rope
- increases score
- increases streak

Wrong answer:
- rope pulled by opponent
- streak reset

Win when rope reaches threshold.

---

# Multiplayer Flow

1. Create room
2. Share link
3. Opponent joins
4. Turn-based answering
5. Winner determined by rope position
